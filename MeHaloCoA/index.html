MeHaloCoA

Halogenated (Cl & Br) compounds search in LC_hrMS

Download source code as a .zip file Download source code as a tar.gz file

MeHaloCoA is an open source R package

that fasten the detection of Cl or Br in LC-hrMS data by using their distinct isotpoic profil and their mass defect

Finding Cl or Br containing compounds in LC-MS data is "simple" you just have to see if the isotopic profil match Cl or Br. But making that annotation manually is time consuming. MeHaloCoA simplifies the process of manual annotation providing a higher-quality peak list in less time.
